## 3DMAKERZ 경매 프로젝트 (Web,  Kiri:Moto)


## 목적
- 사용자가 3D 프린터의 도면을 경매에 올리면 파트너사들이 가격, 납품일을 입력해서 경매에 참여
- 경매, 결제, 제작 진행을 공유하는 관리 페이지 

## 담당
- 디자인, 웹퍼블리싱 제외 1인개발

## 기능 : 올메이크
### 고객이 제작 도면을 업로드하고 견적을 문의하는 사이트
- 절삭 / 레이저 / 3D프린팅 중 가공 방법 선택
